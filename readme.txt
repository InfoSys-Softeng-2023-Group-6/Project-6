=============================================
            Project-6 Readme
=============================================

1. **Project Overview:**
    - This UiPath project automates the scraping of company data from the 'Companies Office' and 'OpenCorporates' websites

2. **Prerequisites:**
    - UiPath Studio installed.
    - You have FireFox with the UiPath extension installed

3. **Setup Instructions:**
    a. Clone repository from here if you don't already have access to the files
    b. Open a new project in UiPath Studio
    c. Navigate inside the project-6 folder and open the project.json file

4. **Run Instructions:**
    a. Click the 'Project' tab on the right of the UiPath window
    b. Open 'main-sequence.xaml'
    c. Select 'Run File'
    d. When the scraping is complete, it will output the file in the project base folder (Not the UiPath project folder)

5. **Configuration:**
   - Ensure that there is a "company_list.xlsx" file that list the company names in the first column
   - This file can be configured to a custom list of company names
